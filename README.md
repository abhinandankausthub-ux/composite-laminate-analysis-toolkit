## Features

- Classical Lamination Theory (CLT) laminate solver
- ABD stiffness matrix computation
- Ply stress analysis
- Failure prediction using:
  - Maximum Stress
  - Tsai-Hill
  - Tsai-Wu
- Progressive ply failure simulation
- Laminate stacking sequence visualization
- Stress distribution through thickness
- Laminate stacking sequence optimization
- Automated engineering report generation
